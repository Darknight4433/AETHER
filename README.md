# AETHER — Personal AI Operating System

A real-time, event-driven autonomous platform mapping cognitive logic models into native OS environments.

## Features:
- **Voice AI (real-time)**: Seamless, interruption-capable speech loops utilizing locally hosted LLMs and conversational engines.
- **Screen Awareness**: Context-aware pipeline that captures, translates, and assesses active desktop elements.
- **Multi-step Planning Agent**: Synthesizes complex user commands into strict JSON actions utilizing a multi-model tool routing pipeline.
- **OS Control (Sandboxed)**: Robust, ring-fenced `AETHER_SANDBOX` enabling file structure edits, application spawning, OS macro automation, and web searching.
- **Autonomous Suggestions**: Ambient telemetry processing environment blocks and predicting requisite workflows based on historic telemetry.
- **Memory + Habit Learning**: Cognitive engine matching temporal action correlations to user metrics—seamlessly updating preference schema maps organically.
- **Live UI Dashboard**: Asynchronous FastApi broadcast mapping live logic traces, cognitive loops, and execution logs dynamically into a React-equivalent glassmorphism dashboard.

## Tech Stack:
- **Python** (Core Runtime Engine)
- **Ollama** (Local Generative Models)
- **FastAPI / Uvicorn** (Asynchronous Dashboard Microservice)
- **HTML / Vanilla CSS** (Live Metrics Dashboard)
- **Deepgram / ElevenLabs** (Transcriptions & Neural Voicing)

## Demo
Please run `python main.py` to trigger the `AetherController`. Aether will instantiate ambient background telemetry traces natively, while concurrently mounting the visual operations dashboard on port 8000. Open `ui/index.html` locally to experience real-time neural tracking logic arrays bridging Voice OS outputs to action handlers.
